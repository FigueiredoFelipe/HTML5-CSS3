Main formatting methods

Bold

Testing bold using the B tags. It's not semantic.

Now we are goin to use a semantic way, using the strong tags.

Italic

To apply emphasis in the text we can use antiquate I tag .

If we select the text we want to emphasize,we can use control + shift + P, and choose Wrap with abbreviation, than selecting the "i", to put tag in the hole selection.

Another and a semantic way, is to use the tag EM.

Marked text

We can mark the text using the tag MARK.

Big and small texts:

We are creating one big and one small text in the paragraph.

Deleted text:

We can mark a text like excluded, or deleted. I can be read but not to be considered.

Inserted text:

We mark a text as inserted using the tag ins. The tag U also works, but it's not semantic.

Envelope a text:

We can envelope a text using the tag SUP to read better the equation x44 + 3 for instance.

Subscribed text:

We can subscribe a text using the tag SUB to write H2O
